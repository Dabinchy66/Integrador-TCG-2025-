INSTRUCCIONES DE USO

El juego es ejecutable de 2 maneras, una descargandolo, se debe primero descargar el juego en zip, descomprimirlo y ejecutar el index.html asi empieza a correr el juego o sino esta la otra la cual es ejecutable desde git pages. Para que pueda correr se utiliza un css, js y HTML. Todo corre sobre un index.html el cual una vez se abre es modificado por el JavaScript. Para poder descargar el juego se debe descargar desde el git hub el cual trae todos los demas archivos junto a carpetas las cuales son fundamentales para que corra este juego, como si fuese una computadora o un sistema. ¿Dónde puedo jugar este juego? Se puede jugar únicamente en computadora
